# Legacy Documentation

import CardWithImage from '/src/theme/CardWithImage';
import { useThemeConfig } from '@docusaurus/theme-common';

export const Cards = () => {
  const { cards } = useThemeConfig();
  return (
    <div className='row'>
      {cards.map((props, idx) => (
        <CardWithImage key={idx} {...props} />
      ))}
    </div>
  );
};

The IOTA legacy documentation is no longer available. Any documentation which is still valid will be ported to the
[IOTA Wiki](/), the one source of truth for all the IOTA documentation.
